Orange Hover Button

Dieser Button zeigt einen orangefarbenen Hover-Effekt, der ansprechendes Feedback gibt, wenn der Benutzer ihn anklickt.

HTML

                
<button class="demo-button">Klicke mich</button>

CSS

                
.demo-button {
    background-color: #EE6B4D;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 16px;
}

.demo-button:hover {
    background-color: #9BC0D9;
}
